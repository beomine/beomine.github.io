## 역대 최대 모델, 유출로 세상에 드러나다

2026년 3월 26일, Fortune이 단독 보도 하나를 터뜨렸다. Anthropic의 내부 콘텐츠 관리 시스템(CMS) 설정 오류로 약 **3,000개의 미공개 자산**이 암호화 없이 공개 검색 가능한 상태로 노출된 것이다.

그 안에는 **Claude Mythos** — Anthropic이 "역대 가장 강력한 AI 모델"이라고 부르는 신규 모델의 초안 발표문, 벤치마크 데이터, 내부 문서가 포함되어 있었다.

며칠 뒤에는 **Claude Code의 소스코드까지 유출**되는 2차 보안 사고가 발생했다. 한 주 만에 두 번의 대형 유출. 일각에서는 "역대 최고의 PR 스턴트 아니냐"는 농담도 나왔다.

## Claude Mythos란?

### 핵심 스펙

| 항목 | 내용 |
|------|------|
| 모델명 | Claude Mythos (내부 코드명) |
| 티어 | **Capybara** (Opus 위의 새로운 최상위 티어) |
| 파라미터 | **10조 (10 Trillion)** |
| 아키텍처 | MoE(Mixture of Experts) 추정 |
| 특화 분야 | 사이버보안, 복잡한 코딩, 학술 추론 |
| 현재 상태 | 얼리 액세스 (사이버보안 고객 한정) |
| 정식 출시 | 미정 (6월 말 예측 54%) |

### Anthropic 모델 계보

```
Haiku (경량)
  ↓
Sonnet (균형)
  ↓
Opus (최상위) ← 현재 공개된 최강 모델
  ↓
Capybara (신규) ← Claude Mythos가 여기
```

**Capybara**는 Opus 위에 새로 추가된 티어다. Opus가 현재 최상위라면, Capybara는 **그 위의 초프리미엄 등급**이다.

## 10조 파라미터의 의미

### 모델 크기 비교

| 모델 | 파라미터 수 | 출시 |
|------|:---:|:---:|
| GPT-3 | 1,750억 | 2020 |
| GPT-4 | 1.8조 (추정, MoE) | 2023 |
| Claude Opus 4.6 | 비공개 | 2025 |
| Llama 3 405B | 4,050억 | 2024 |
| **Claude Mythos** | **10조** | 2026 |

10조 파라미터는 GPT-3의 약 **57배**, GPT-4 추정치의 약 **5.5배**에 해당한다.

### MoE 아키텍처

10조 파라미터를 Dense(밀집) 모델로 운영하는 것은 현재 기술로 거의 불가능하다. 전문가들은 Claude Mythos가 **MoE(Mixture of Experts)** 아키텍처를 사용할 것으로 추정한다.

MoE는 전체 파라미터 중 **일부 전문가 네트워크만 활성화**하여 추론하는 방식이다.

| | Dense 모델 | MoE 모델 |
|---|---|---|
| 추론 시 활성 파라미터 | 전체 | 일부 (10~20%) |
| 추론 비용 | 매우 높음 | 상대적으로 낮음 |
| 학습 비용 | 높음 | 매우 높음 |
| 전문성 | 범용 | 영역별 전문가 조합 |

10조 파라미터 MoE라면, 추론 시 실제 활성화되는 파라미터는 1~2조 수준일 수 있다. 그래도 현존 최대 규모다.

## 특화 분야: 사이버보안, 코딩, 학술 추론

유출된 내부 문서에 따르면, Mythos는 세 가지 영역에서 **Claude Opus 4.6을 명확히 초과**한다.

### 1. 사이버보안

Anthropic 내부 문서에서 Mythos의 사이버보안 능력을 **"전례 없는 수준"**이라고 표현했다. 이것은 강점이자 동시에 가장 큰 우려 사항이다.

**방어 측면:**
- 코드 취약점 자동 탐지 및 패치 제안
- 네트워크 침입 패턴 실시간 분석
- 보안 감사 자동화

**우려 사항:**
- 공격에도 활용될 수 있음
- "방어자가 대응하는 속도보다 공격이 빠르게 확산될 수 있다"는 경고
- 이것이 Anthropic이 출시를 신중하게 진행하는 핵심 이유

### 2. 복잡한 코딩

단순 코드 생성을 넘어, **대규모 코드베이스의 아키텍처 설계와 리팩토링**에서 능력 향상이 보고되었다. 이전 모델에서 어려웠던 멀티 파일 수정, 시스템 간 의존성 파악 등이 개선되었을 것으로 추정된다.

### 3. 학술 추론

수학, 과학, 논리적 추론에서의 성능 향상. 복잡한 다단계 추론 문제에서 이전 모델 대비 유의미한 개선이 있는 것으로 알려졌다.

## 왜 아직 출시하지 않는가?

Anthropic은 Mythos를 **의도적으로 느리게 출시**하고 있다. 유출 문서에서 그 이유를 찾을 수 있다.

### 이유 1: 비용

> "우리가 서빙하기에 매우 비싸고, 고객에게도 매우 비쌀 것이다"
> — 유출된 내부 초안

현재 Claude Opus 4.6의 API 가격:

| | 입력 | 출력 |
|---|---|---|
| Opus 4.6 | $15 / 100만 토큰 | $75 / 100만 토큰 |
| Mythos (예상) | **훨씬 비쌀 것** | **훨씬 비쌀 것** |

정식 출시 전에 **경량화(distillation), 양자화(quantization), 추론 최적화**가 선행되어야 한다.

### 이유 2: 안전성

사이버보안 능력이 너무 강해서, 오용 시 방어보다 공격이 더 빨라질 수 있다는 우려가 있다. Anthropic은 사이버 방어 기업에 먼저 제공하여 **방어 체계를 구축한 후 일반 공개**하려는 전략이다.

### 이유 3: 책임 있는 출시

Anthropic의 Responsible Scaling Policy(RSP)에 따라 안전성 평가를 통과해야 출시할 수 있다. 기존 모델보다 능력이 크게 향상되었기에 **평가 기준도 더 엄격**해진다.

## AI 경쟁 구도에 미치는 영향

### 현재 최상위 모델 비교

| 기업 | 최상위 모델 | 특징 |
|------|-----------|------|
| Anthropic | **Mythos (Capybara)** | 10조 파라미터, 사이버보안·코딩 특화 |
| OpenAI | GPT-5 계열 | IPO 준비 중, 상세 미공개 |
| Google | Gemini Ultra 2 | 멀티모달 강점 |
| Meta | Llama 4 | 오픈소스, 405B |
| xAI | Grok-3 | 실시간 데이터 접근 |

### Anthropic의 전략적 의미

Mythos는 Anthropic이 **"가장 안전한 AI 기업"에서 "가장 강력한 AI 기업"**으로도 인정받으려는 시도다. 기존에는 OpenAI·Google 대비 모델 성능에서 뒤처진다는 인식이 있었지만, Mythos로 그 격차를 단번에 뒤집으려 한다.

다만 이는 양날의 검이다. "안전 우선"을 표방하면서 "역대 최강 사이버보안 모델"을 만든 것이 모순 아니냐는 비판도 있다.

## 개발자가 알아야 할 것

### 1. Capybara 티어 = 프리미엄 가격

정식 출시 시 **Opus보다 훨씬 비싼 가격**이 예상된다. 모든 작업에 Mythos를 쓰는 것은 비현실적이고, 복잡한 추론이 필요한 작업에만 선별적으로 사용하는 전략이 필요하다.

### 2. 모델 라우팅의 중요성

```
간단한 작업 → Haiku (저렴, 빠름)
일반 작업   → Sonnet (균형)
복잡한 작업 → Opus (고성능)
최고난도    → Mythos/Capybara (최상위)
```

작업 복잡도에 따라 **자동으로 모델을 선택하는 라우팅 전략**이 더 중요해진다.

### 3. 사이버보안 도구 생태계 변화

Mythos의 보안 능력이 공개되면, **AI 기반 보안 도구 시장이 급성장**할 것이다. 코드 보안 감사, 취약점 스캐닝, 침투 테스트 분야에서 새로운 도구들이 쏟아질 수 있다.

### 4. Claude Code와의 통합 가능성

현재 Claude Code는 Opus 4.6을 사용한다. Mythos가 정식 출시되면 Claude Code에서도 Capybara 티어를 선택할 수 있을 것이다. 특히 대규모 리팩토링이나 보안 감사 같은 작업에서 큰 차이를 보일 수 있다.

## 마무리

Claude Mythos는 **AI 모델 스케일링의 새로운 이정표**다.

핵심 정리:

- **10조 파라미터** — 현존 최대 규모, MoE 아키텍처 추정
- **Capybara 티어** — Opus 위의 새로운 최상위 등급
- **사이버보안·코딩·추론 특화** — 강점이자 우려 사항
- **출시 미정** — 비용 최적화와 안전성 평가 진행 중
- **6월 말 출시 유력** — Polymarket 54% 확률

유출이라는 해프닝으로 세상에 알려졌지만, Mythos의 존재 자체가 AI 경쟁의 새로운 국면을 열었다. Anthropic이 이 모델을 어떻게 출시하느냐가 2026년 하반기 AI 산업의 향방을 결정할 것이다.
